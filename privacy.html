서비스 개요 및 운영자 정보
Section 01

Rainy Note는 유리창에 빗방울, 서리, 안개, 물방울 효과를 배경으로 손가락으로 닦으며 메시지와 그림을 남기는 감성 모바일 웹 애플리케이션입니다.

운영자 정보

서비스명: Rainy Note

유형: 개인 운영 웹 애플리케이션 (무료 서비스)

접근 방식: 웹 브라우저 (별도 설치 불필요, 회원가입 불필요)

문의: 아래 제10조 참고

Rainy Note는 완전 무료로 제공되며, 회원가입·로그인이 전혀 필요하지 않습니다. 서비스 운영 비용은 앱 내 구글 애드센스 광고 수익으로 충당됩니다.

광고 시청 횟수·날짜 앱 내 자동 기록 일일 광고 한도 관리 기기 로컬(localStorage)

수집하지 않는 정보
Section 03

Rainy Note는 아래 정보를 절대 수집하지 않습니다.

  • 이름, 이메일 주소, 전화번호 등 개인 식별 정보
  • 회원가입 또는 로그인 정보 (계정 자체가 없습니다)
  • 위치 정보 (GPS, IP 기반 위치 등)
  • 카메라·마이크로 촬영·녹음된 미디어 (업로드된 사진은 기기에서 직접 처리됩니다)
  • 결제 정보 (유료 서비스 없음)
  • 연락처, 캘린더, 파일 등 기기 내 데이터
  • 앱 서버로 전송되는 사용자 생성 콘텐츠

로컬 저장소(localStorage) 사용
Section 04

Rainy Note는 서버 데이터베이스 없이 브라우저의 localStorage만을 사용하여 기기 내에서 데이터를 관리합니다.

저장되는 키 목록

rn_points — 현재 빗방울(포인트) 잔액

rn_ads_today — 오늘 광고 시청 횟수

rn_ads_date — 일일 초기화 기준 날짜

rn_gallery_v1 — 저장된 이미지 썸네일 배열 (최대 50개)

rn_stats_v1 — 광고·저장 통계 (최근 90일)

  • 모든 데이터는 사용자 기기 내에만 존재합니다.
  • 외부 서버로 전송되지 않습니다.
  • 브라우저 설정 → 사이트 데이터 삭제로 완전 제거 가능합니다.
  • 다른 기기나 브라우저와 자동으로 동기화되지 않습니다.

Google AdSense 광고
Section 05

Rainy Note는 서비스 운영 비용 충당을 위해 Google LLC의 AdSense 광고 서비스를 사용합니다. 광고는 앱 내 포인트 충전 모달에서 표시됩니다.

Google AdSense는 사용자의 관심사에 맞는 맞춤형 광고를 제공하기 위해 쿠키(Cookie) 및 광고 식별자를 사용할 수 있습니다. 이는 Google의 광고 시스템이 자동으로 처리하며, Rainy Note 운영자는 해당 데이터에 접근하거나 보유하지 않습니다.
Google AdSense 주요 사항

게시자 ID: ca-pub-1450948725343408

광고 슬롯: RainyNote_Main (포인트 충전 모달 내)

광고 유형: 자동 형식, 반응형

데이터 처리자: Google LLC (운영자가 아님)

Google의 광고 개인정보 처리 방식에 대한 상세 내용은 아래 링크에서 확인하실 수 있습니다.

  • Google 개인정보처리방침
  • Google 광고 작동 방식
  • Google 광고 개인화 설정 (맞춤 광고 해제 가능)
  • IAB 광고 수신 거부 (aboutads.info)

Section 06

Rainy Note가 사용하는 외부 서비스 목록입니다. 각 서비스는 해당 제공자의 개인정보처리방침을 따릅니다.

서비스 제공자 목적 데이터 전송
Google AdSense Google LLC 광고 표시 및 수익화 광고 쿠키·식별자
Google Fonts Google LLC 웹 폰트(Inter) 로드 IP 주소 (일시적)

아동 개인정보 보호
Section 07

Rainy Note는 만 14세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다.

  • 앱 자체는 연령 제한 없이 이용 가능합니다.
  • 계정 생성이 없으므로 연령 확인 절차도 없습니다.
  • 아동의 개인정보를 수집·저장하지 않습니다.

Google AdSense는 맞춤형 광고와 관련하여 자체적인 아동 보호 정책을 운영합니다. 자세한 내용은 Google AdSense 아동 보호 정책을 참고하세요.

이용자의 권리
Section 08

Rainy Note는 서버에 개인정보를 보유하지 않으므로 이용자는 언제든지 스스로 데이터를 관리할 수 있습니다.

  • 데이터 확인: 브라우저 개발자 도구 → Application → localStorage에서 직접 확인 가능
  • 데이터 삭제: 앱 내 Reset 기능 또는 브라우저 사이트 데이터 삭제로 즉시 제거 가능
  • 광고 맞춤화 거부: Google 광고 설정에서 맞춤 광고 비활성화 가능
  • 쿠키 차단: 브라우저 설정에서 서드파티 쿠키를 차단하면 광고 맞춤화가 제한됩니다
서버에 저장된 개인정보가 없으므로, 별도의 삭제 요청 절차 없이 브라우저 데이터 삭제만으로 완전한 데이터 제거가 가능합니다.

방침 변경 안내
Section 09

본 개인정보처리방침은 법령, 서비스 변경, 광고 정책 변경 등에 따라 업데이트될 수 있습니다.

  • 방침 변경 시 본 페이지 상단의 최종 수정일을 업데이트합니다.
  • 중요한 변경이 있을 경우 앱 내 토스트 메시지로 안내할 수 있습니다.
  • 변경 후 앱을 계속 이용하면 변경된 방침에 동의한 것으로 간주됩니다.
